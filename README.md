# The Last Game For Friends

Кооперативная игра про выживание с друзьями в мире, захваченном зомби.

## Структура проекта

```
the-last-game-for-friends/
│
├── client/           # Клиентская часть
├── server/           # Серверная часть
├── shared/           # Общие файлы
├── package.json
└── README.md
```

## Установка и запуск

1. Установите зависимости:
```bash
npm install
```

2. Запустите сервер:
```bash
npm run dev
```

3. Запустите клиент (в отдельном терминале):
```bash
npm run client
```

## Особенности игры

- Кооперативный геймплей
- Система строительства базы
- Бои с зомби
- Прокачка персонажей
- Сетевая игра в реальном времени
